# LeiloesTDSat

## Descrição
Sistema informatizado desenvolvido em Java para apoiar as rotinas de uma casa de leilões, permitindo o cadastro, listagem e controle de produtos.

## Tecnologias utilizadas
- Java
- MySQL
- Git
- GitHub

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Atualização de status (A Venda / Vendido)
- Integração com banco de dados MySQL
